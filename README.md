**Project Title:**

Sales Data Analysis Project 

**Project Description:**

This is a Data Analytics Project in which the sales dataset was analyzed using Excel, SQL, and Power BI to generate meaningful business insights and recommendations.
In this project:
•	Excel was used for data cleaning and preparation
•	SQL was used to analyse customer, product, and sales data
•	Power BI was used to create an interactive dashboard
•	The insights report was created to help improve business performance

**Project Files:**

1.	task 1 and 2 .xlsx → Dataset and Excel analysis
2.	Task 3 .sql → SQL queries used for analysis
3.	Task 4 .pbix → Power BI Dashboard file
4.	Insights Report → Business insights and recommendations

**How to Run This Project:**

Follow the steps below:

**Run Excel File**

1.	Open the Excel file:
task 1 and 2 .xlsx
2.	Review dataset
3.	Check analysis

**Run SQL File**

1.	Open MySQL or SQL Server
2.	Create a database:

3.	Create orders table
4.	Import dataset
5.	Run queries from the file: Task 3 .sql

**Run Power BI Dashboard**

3.	View dashboard
